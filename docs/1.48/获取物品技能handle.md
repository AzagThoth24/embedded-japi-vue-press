# 获取物品技能 handle

```jass
function GetItemAbility takes item Item, integer slot returns ability

```

# 描述

获取物品技能的 handle 返回值 可以用在 ydjapi 的技能接口

# 参数

| 类型 | 名字 | 说明              |
| ---- | ---- | ----------------- |
| 物品 | Item | 指定物品          |
| 整数 | slot | 指定槽位 0 从开始 |

# 返回值

| 类型    | 说明 |
| ------- | ---- |
| ability | 技能 |
